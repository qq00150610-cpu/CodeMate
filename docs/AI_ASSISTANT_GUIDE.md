# AI 助手使用指南

## 一、功能概述

CodeMate AI 助手是集成在 IDE 中的智能编程辅助工具，基于大语言模型技术，为开发者提供代码解释、优化、调试等功能。

## 二、支持模型

### 2.1 OpenAI 系列

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| GPT-4o | 最新最强，支持多模态 | 复杂任务、代码解释 |
| GPT-4 Turbo | 快速高效 | 日常编程辅助 |
| GPT-3.5 Turbo | 性价比最高 | 简单任务、快速问答 |

### 2.2 Claude 系列

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| Claude 3 Opus | 最强大 | 深度代码分析 |
| Claude 3 Sonnet | 平衡性能 | 日常开发 |
| Claude 3 Haiku | 快速响应 | 简单问答 |

### 2.3 Ollama 本地模型

| 模型 | 特点 | 说明 |
|------|------|------|
| Llama 3 | 开源大模型 | 需要本地安装 |
| Code Llama | 代码专用 | 适合代码生成 |
| Mistral | 高效开源 | 性能优秀 |

## 三、快捷指令

### 3.1 功能说明

| 快捷指令 | 功能 | 说明 |
|----------|------|------|
| 🔍 解释代码 | 分析代码功能 | 提供代码逻辑说明 |
| 🔧 重构建议 | 优化代码结构 | 改进代码质量 |
| 📝 添加注释 | 生成代码注释 | 自动添加中文注释 |
| 🐛 查找Bug | 检查潜在问题 | 发现代码隐患 |
| ✅ 生成测试 | 生成单元测试 | 自动生成测试用例 |
| 🌐 翻译代码 | 代码语言转换 | 转换到其他语言 |

### 3.2 使用方法

1. 选中要处理的代码
2. 点击快捷指令按钮
3. AI 将根据选中代码生成响应

## 四、API Key 配置

### 4.1 OpenAI API Key

1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 注册账号并充值
3. 在 API Keys 页面创建新的 API Key
4. 在 CodeMate 设置中配置 API Key

### 4.2 Claude API Key

1. 访问 [Anthropic Console](https://console.anthropic.com/)
2. 注册账号并订阅
3. 创建 API Key
4. 在 CodeMate 设置中配置

### 4.3 Ollama 本地部署

1. 安装 Ollama
   ```bash
   # macOS/Linux
   curl -fsSL https://ollama.com/install.sh | sh
   
   # 或下载安装包
   ```

2. 下载模型
   ```bash
   ollama pull llama3
   ollama pull codellama
   ```

3. 启动服务
   ```bash
   ollama serve
   ```

4. 在 CodeMate 中选择 Ollama 模型

## 五、使用技巧

### 5.1 获得更好的回答

1. **选中相关代码**：处理前先选中要分析的代码
2. **描述具体问题**：问题描述越具体，回答越准确
3. **分步提问**：复杂问题可以分步提问
4. **查看代码上下文**：AI 可以利用当前文件上下文

### 5.2 响应中的代码操作

- **插入代码**：点击"插入"按钮将代码插入编辑器
- **复制代码**：点击"复制"按钮复制到剪贴板
- **代码高亮**：支持多种语言的语法高亮

### 5.3 对话历史

- 对话历史自动保存在本地
- 支持导出对话为 Markdown
- 可以清除历史记录

## 六、安全提示

1. **API Key 安全**：
   - 不要将 API Key 提交到代码仓库
   - 定期更换 API Key
   - 使用环境变量存储敏感信息

2. **隐私保护**：
   - 敏感代码可使用本地 Ollama 模型
   - 确认 API 提供商的隐私政策

## 七、常见问题

### Q1: API 调用失败？

**可能原因：**
- API Key 无效或过期
- 网络连接问题
- API 配额用尽

**解决方法：**
1. 检查 API Key 是否正确
2. 确认网络连接
3. 查看 API 提供商的使用限制

### Q2: 响应速度慢？

**可能原因：**
- 网络延迟
- 模型处理时间长
- 服务器负载高

**解决方法：**
1. 使用本地 Ollama 模型
2. 选择更快的模型（如 GPT-3.5）
3. 稍后重试

### Q3: 如何切换模型？

在 AI 设置页面，选择不同的模型即可。切换后立即生效。
