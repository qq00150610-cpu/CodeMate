## 简介

CodeMate 是一款运行在 Android 上的集成开发环境（IDE），深度集成了阿里云百炼 AI 编程助手，为移动端开发者提供接近桌面级的编码体验。

### 主要特性

- 🤖 **AI 编程助手**：基于阿里云通义千问，智能代码解释、优化、调试
- 📱 **原生移动体验**：针对移动设备优化的键盘和手势操作
- 💻 **完整 VS Code 功能**：基于 code-server，支持 VS Code 扩展
- 🔧 **完全离线运行**：内置 Termux 环境
- ⚡ **PRoot Ubuntu 环境**：运行在 Ubuntu 环境下的 code-server

## 安装应用

### 方法一：从 APK 安装

1. 下载最新的 `app-release.apk` 文件
2. 打开手机设置 → 安全 → 允许「未知来源应用」
3. 点击 APK 文件开始安装

### 方法二：从 GitHub 下载

访问 [GitHub Releases](https://github.com/your-username/CodeMate/releases) 下载最新版本。

## 基础使用

### 1. 启动应用

首次启动会初始化 Ubuntu 环境，请耐心等待。

### 2. 创建或打开项目

- 点击「打开文件夹」选择现有项目
- 点击「新建项目」从模板创建
- 支持的文件类型：所有编程语言文件

### 3. 编辑代码

- 点击文件在编辑器中打开
- 使用下方的快捷键栏输入特殊字符
- 双指缩放调整字体大小
- 长按弹出上下文菜单

### 4. 快捷键栏

底部快捷键栏提供常用编程符号：

| 按钮 | 输入 |
|------|------|
| Tab | Tab 缩进 |
| {} | 花括号 |
| () | 圆括号 |
| [] | 方括号 |
| "" | 引号 |
| ; | 分号 |

## AI 助手

### 开启 AI 功能

1. 点击右上角的 AI 图标打开侧边栏
2. 首次使用需要配置后端服务地址（联系管理员获取）

### 使用快捷指令

选中代码后，使用侧边栏的快捷指令：

- 📖 **解释代码**：详细解释选中代码的功能
- ⚡ **优化代码**：提供性能优化建议
- 🐛 **调试代码**：分析并修复 Bug
- 🧪 **生成测试**：生成单元测试用例
- 🔄 **翻译代码**：转换为其他语言
- 🏗️ **重构代码**：改善代码结构

### 自定义提问

直接在输入框中输入问题，AI 会根据上下文给出回答。

### 支持的模型

| 模型 | 特点 |
|------|------|
| 通义千问-超快速 | 快速响应，适合日常对话 |
| 通义千问-均衡 | 平衡性能与成本 |
| 通义千问-最强 | 最强能力，适合复杂任务 |
| 通义灵码 | 代码生成、分析专家 |

## 高级功能

### 1. 项目模板

从模板快速创建项目：

1. 点击「新建项目」
2. 选择模板类型
3. 填写项目信息
4. 点击「创建」

### 2. 键盘适配

应用会自动检测外部键盘并调整布局：

- 检测到键盘时隐藏底部快捷键栏
- 支持快捷键组合（Ctrl+C, Ctrl+V 等）
- 自动调整边距

### 3. 性能模式

根据设备性能自动调整：

- **普通模式**：完整功能体验
- **低内存模式**：降低画质，限制历史记录
- **省电模式**：禁用动画，降低刷新率

### 4. 主题设置

支持浅色/深色/跟随系统三种主题。

### 5. 更新检查

应用会自动检查更新：

- 设置 → 检查更新
- 可开启「仅 Wi-Fi 下载」节省流量
- 可开启「接收测试版」体验新功能

## 常见问题

### Q: 首次启动很慢？

A: 首次启动需要初始化约 500MB 的 Ubuntu 环境，请连接 Wi-Fi 并耐心等待。

### Q: AI 功能无法使用？

A: 请检查：
1. 网络连接是否正常
2. 后端服务地址是否正确
3. 联系管理员确认服务状态

### Q: 内存不足怎么办？

A: 
1. 关闭不需要的项目
2. 设置 → 开启「低内存模式」
3. 清除缓存

### Q: 如何导入文件？

A:
1. 使用文件管理器将文件放入 `/storage/emulated/0/Download/CodeMate/`
2. 在应用内点击「刷新」同步文件
3. 或使用 ADB 推送：`adb push local/file /sdcard/CodeMate/project/`

### Q: 支持哪些编程语言？

A: CodeMate 基于 code-server，理论上支持 VS Code 支持的所有语言，包括：
- Kotlin, Java, Swift
- Python, JavaScript, TypeScript
- C, C++, Go, Rust
- HTML, CSS, SQL
- 等等...
