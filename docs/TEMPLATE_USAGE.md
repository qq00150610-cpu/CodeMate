# 模板使用指南

## 一、概述

CodeMate 提供丰富的项目模板，帮助开发者快速创建新项目。支持 Android、Flutter、Web、后端等多种类型。

## 二、模板分类

### 2.1 Android 开发

| 模板名称 | 说明 |
|----------|------|
| Empty Android App | 基础的 Android 应用，使用 Kotlin |
| MVVM Architecture | 采用 MVVM 架构的模板 |

### 2.2 Flutter 开发

| 模板名称 | 说明 |
|----------|------|
| Flutter Basic App | 基础的 Flutter 应用 |
| Provider Architecture | 使用 Provider 状态管理 |

### 2.3 Web 开发

| 模板名称 | 说明 |
|----------|------|
| HTML/CSS/JS Basic | 基础的 HTML5 网站 |
| React App | React + Vite 项目模板 |

### 2.4 后端开发

| 模板名称 | 说明 |
|----------|------|
| Python Flask API | Python Flask REST API |
| Node.js Express API | Node.js Express REST API |
| Go HTTP Server | Go HTTP 服务器 |

### 2.5 其他

| 模板名称 | 说明 |
|----------|------|
| Markdown Document | Markdown 文档模板 |
| Shell Script Collection | 常用 Shell 脚本 |

## 三、使用流程

### 3.1 选择模板

1. 点击「新建项目」
2. 浏览模板分类
3. 选择需要的模板
4. 查看模板预览

### 3.2 配置项目

1. 输入项目名称
2. 选择保存位置
3. 可选：自定义配置

### 3.3 创建项目

1. 点击「创建」
2. 等待模板生成
3. 项目自动打开

## 四、自定义模板

### 4.1 保存为模板

1. 完成项目开发
2. 点击「另存为模板」
3. 填写模板信息
4. 选择保存位置

### 4.2 导入/导出

1. 导出：选择模板 → 导出为 .zip
2. 导入：导入 .zip 文件

### 4.3 模板存储

模板保存在：
```
/storage/emulated/0/CodeMate/Templates/
```

## 五、模板预览

每个模板支持预览：
- 文件结构
- 主要代码文件
- 依赖列表
- 使用说明

## 六、快捷操作

### 6.1 快速创建

在项目列表页面：
- 长按模板快速创建
- 拖拽调整顺序

### 6.2 收藏模板

- 点击收藏按钮
- 收藏模板显示在顶部

### 6.3 搜索模板

- 输入关键词搜索
- 支持按标签筛选
