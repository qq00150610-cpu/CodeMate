# 更新系统说明

## 一、更新机制

### 1.1 版本检查

CodeMate 通过以下方式检查更新：
- 启动时自动检查
- 手动检查更新
- 定时后台检查（每24小时）

### 1.2 更新来源

- GitHub Releases: https://github.com/vhqtvn/VHEditor-Android/releases
- 当前版本：2.22.0

## 二、更新流程

### 2.1 发现新版本

当有新版本可用时：
1. 设置页面显示更新提示
2. AppBar 图标显示徽章
3. 可查看更新日志

### 2.2 下载更新

1. 点击更新按钮
2. 确认 WiFi 环境（如设置了仅 WiFi 下载）
3. 后台下载 APK
4. 显示下载进度

### 2.3 安装更新

1. 下载完成后弹出安装提示
2. 点击安装
3. 系统安装界面
4. 完成更新

## 三、设置选项

### 3.1 自动检查更新

- 默认开启
- 每次启动检查一次
- 后台定时检查（每24小时）

### 3.2 仅 WiFi 下载

- 默认开启
- 节省移动数据
- WiFi 环境下自动下载

### 3.3 更新渠道

| 渠道 | 说明 |
|------|------|
| Stable | 稳定版，正式发布 |
| Beta | 测试版，可能不稳定 |

## 四、断点续传

### 4.1 自动重试

下载失败时自动重试：
- 最多重试 3 次
- 指数退避策略
- 显示错误原因

### 4.2 手动重试

下载失败后可手动重试：
1. 点击失败提示
2. 选择重试
3. 从断点继续下载

## 五、权限要求

### 5.1 Android 13+ (API 33+)

需要权限：
- `READ_MEDIA_IMAGES` (可选)
- `INSTALL_PACKAGES` (通过 PackageInstaller)

### 5.2 Android 10-12 (API 29-32)

需要权限：
- `REQUEST_INSTALL_PACKAGES`

### 5.3 Android 9 (API 28 及以下)

需要权限：
- `WRITE_EXTERNAL_STORAGE`
- `REQUEST_INSTALL_PACKAGES`

## 六、常见问题

### Q1: 下载失败？

**可能原因：**
- 网络不稳定
- 存储空间不足
- 权限被拒绝

**解决方法：**
1. 检查网络连接
2. 清理存储空间
3. 确认安装权限

### Q2: 无法安装？

**可能原因：**
- 未知来源限制
- 签名不匹配
- 系统版本不兼容

**解决方法：**
1. 在设置中允许安装未知来源
2. 确保签名一致
3. 检查系统版本要求

### Q3: 更新后无法启动？

**可能原因：**
- 安装不完整
- 签名问题
- 数据不兼容

**解决方法：**
1. 卸载后重新安装
2. 清除应用数据
3. 联系开发者支持
